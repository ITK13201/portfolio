# Portfolio

This is an application for my portfolio site.
You can access it at the following link.

https://www.i-tk.dev

## Components

- Framework: [Next.js](https://nextjs.org/)
- Design: [Material UI](https://mui.com/)

## Install

```shell
yarn install
```

## Usage

### Start (Development)

```shell
yarn dev
```

### Start (Production)

```shell
yarn start
```

### Build

```shell
yarn build
```

### Tracking and Format

```shell
yarn watch
```
